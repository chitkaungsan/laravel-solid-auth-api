# Laravel SOLID Auth API

A Laravel authentication API built with SOLID principles.

## Features
- Register
- Login
- Profile
- Update Profile
- Change Password

## Architecture
- Service Layer
- Repository Pattern
- SOLID Principles
- Pest Testing
- Laravel Sanctum Authentication

## Tech Stack
- Laravel
- Pest
- Sanctum
- SQLite Testing

## API Endpoints

| Method | Endpoint |
|------|------|
| POST | /api/register |
| POST | /api/login |
| GET | /api/profile |
| PUT | /api/profile |
| POST | /api/change-password |
